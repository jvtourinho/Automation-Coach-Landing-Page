== Descrição ==

Tema moderno e responsivo para o Automation Coach - Automatize processos, empodere pessoas.

Este tema apresenta um design limpo e profissional com foco em conversão e experiência do usuário.

Recursos principais:
* Design totalmente responsivo
* Otimizado para performance
* Gradientes personalizáveis
* Seções de hero, benefícios e depoimentos
* Animações suaves
* Suporte para imagens destacadas
* Menu de navegação customizável

== Instalação ==

1. Faça upload da pasta 'automation-coach' para o diretório '/wp-content/themes/'
2. Ative o tema através do menu 'Aparência' no WordPress
3. Adicione suas imagens na pasta 'images/':
   - hero-automation.jpg (imagem do hero)
   - testimonial-1.jpg (foto do depoimento 1)
   - testimonial-2.jpg (foto do depoimento 2)
   - testimonial-3.jpg (foto do depoimento 3)

== Personalização ==

Você pode personalizar as cores editando as variáveis CSS no arquivo style.css:

:root {
  --primary: hsl(270, 80%, 60%);
  --primary-glow: hsl(280, 85%, 70%);
  --background: hsl(0, 0%, 100%);
  --foreground: hsl(0, 0%, 5%);
  /* ... outras variáveis */
}

== Navegação Suave ==

O tema inclui navegação suave entre seções. As âncoras funcionam com os seguintes IDs:
- #sobre
- #beneficios
- #depoimentos
- #contato
